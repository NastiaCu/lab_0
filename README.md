Installing `zsh` and `oh my zsh` as a default shell.

```
brew install zsh 
chsh -s /usr/local/bin/zsh
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
```
